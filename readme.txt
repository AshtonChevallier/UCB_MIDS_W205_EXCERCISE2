Instuctions to view stream

1. Ensure PostgreSQL is running on your instance
2. Ensure tweepy and psycopg2 are install (Use pip install as root!)
3. Execute: python db_setup.p
4. Execute: cd ex_2/tweetwordcount
5. Execute: sparse run
6. Kill job when you feel you've streamed enough (^C)
7. Execute: python ex_2/finalresults.py
8. Execute: python ex_2/histogram.py
